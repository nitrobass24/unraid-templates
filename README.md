# Unraid Templates

Unraid Community Applications templates for [SeedSync](https://github.com/nitrobass24/seedsync).

## Installation

### From Community Applications (recommended)

Search for **SeedSync** in the Unraid Community Applications store.

### Manual

1. In the Unraid web UI, go to **Docker** > **Template Repositories**
2. Add this URL: `https://github.com/nitrobass24/unraid-templates`
3. Click **Save**
4. Go to **Add Container** and select **SeedSync** from the template dropdown

## Configuration

After installing, open the SeedSync web UI and configure:

- **Remote server address** and **port**
- **Username** and **password** (or SSH key)
- **Remote path** to sync from

See the [full documentation](https://nitrobass24.github.io/seedsync/) for details.
